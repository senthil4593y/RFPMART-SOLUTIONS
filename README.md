# USWDS-Based PHP / HTML Templates

A collection of ready-to-use, open source PHP and HTML templates built
using the U.S. Web Design System (USWDS). These templates are designed to
help developers rapidly build accessible, standards-compliant websites
with minimal setup time.

## Live Demo

Please open this link to view live demo: https://www.uswebdesignsystem.org/uswds-1/

Current stable release: **v1.0.0**

## Features

- Built on U.S. Web Design System (USWDS)
- Accessibility-first and WCAG-aligned
- Clean, modular HTML and PHP structure
- Easy to customize and extend
- Suitable for government, enterprise, and public-sector style projects
